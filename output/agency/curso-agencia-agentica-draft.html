Programa resumido

Tres sesiones para construir el sistema completo.

Cada sesión combina arquitectura con implementación real sobre tu propio proyecto. Sales con código funcionando, no con apuntes.

014 h

El orquestador: CLAUDE.md y la capa de comandos.

Cómo funciona un orquestador de agentes y por qué CLAUDE.md es el centro de todo. Construyes el tuyo desde cero: servicios, clientes, delegación y reglas de operación.

  • Arquitectura de un sistema agéntico para agencias.
  • Diseño del CLAUDE.md orquestador y estructura de carpetas.
  • Construcción de skills (slash commands) para operación diaria.
  • Sistema de perfiles de cliente y regla de upsell automático.
024 h

MCPs, automatizaciones e integraciones.

Conecta tu orquestador con el mundo real: Notion como CRM, Meta Ads, Google Ads, Canva, n8n para automatizaciones.

  • Instalación y configuración de MCPs (Notion, Meta, Ahrefs, n8n).
  • REST APIs directas: WordPress sin MCP vía Registry de Windows.
  • Flujos n8n conectados a Claude Code: trigger → agente → resultado.
  • Gestión de credenciales sin riesgos (SOP de claves API).
034 h

Sesiones, SOPs y memoria del sistema.

El sistema que no recuerda no escala. Diseña la memoria del agente: cierre estructurado, biblioteca de SOPs y flujo de inventario de funcionalidades.

  • Sistema de sesiones: apertura, cierre y continuidad entre conversaciones.
  • Biblioteca de SOPs: estructura, plantillas y actualización automática.
  • Inventario de funcionalidades como producto vendible.
  • Flujo completo: cliente nuevo → primer resultado entregado.
BONUSAsync

Contenido, campañas y publicación automatizada.

El sistema en modo publicación: genera copies, carruseles y campañas Meta desde el orquestador. Integración con Vibiz para publicar en redes con un comando.

  • Skills de campaña, copy y auditoría de cuentas publicitarias.
  • Integración Vibiz: publicación social desde Claude Code.
  • Pipeline de contenido: viral detectado → adaptado → publicado.
  • Reportes semanales automatizados por cliente.
