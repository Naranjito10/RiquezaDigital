## Notas para el diseñador / creativos

- Imágenes sugeridas: postre en primer plano con arándanos frescos, colores morados/azules vibrantes
- Para variante A: imagen evocadora, lifestyle (persona disfrutando)
- Para variante B: imagen de ingredientes / producto close-up
- Para variante C: imagen con badge "NUEVO" o "NEW" superpuesto
- Formato recomendado: 1:1 (feed) + 9:16 (stories/reels)
