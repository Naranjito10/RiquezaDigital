Incorpora un nuevo cliente a Riqueza Digital.

1. Pide el nombre del cliente
2. Crea carpeta `clients/<nombre>/`
3. Copia plantilla desde `clients/_template/profile.md`
4. Guía al usuario para completar el perfil: negocio, audiencia, objetivos, KPIs, presupuesto, credenciales
5. Crea las carpetas de trabajo del cliente `clients/<nombre>/marketing/meta/` y `clients/<nombre>/marketing/google/`
6. Crea el archivo de control de horas `clients/<nombre>/imputacion-horas.md` copiando la plantilla desde `clients/_template/imputacion-horas.md`
7. Actualiza el listado de clientes en el CLAUDE.md raíz
8. Confirma que las credenciales se guardan en `.env` y NO en el profile
