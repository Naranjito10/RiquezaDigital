Crea una nueva campaña publicitaria para el cliente.

1. Pregunta qué cliente si no está en contexto
2. Lee `clients/<cliente>/profile.md` para entender el negocio y audiencia
3. Activa el agente Gestor de Anuncios (`agents/ads/CLAUDE.md`)
4. Recoge información necesaria: plataforma, objetivo, presupuesto, fechas, audiencia, creativos
5. Muestra plan completo (campaign + adsets + ads) ANTES de ejecutar
6. Espera confirmación explícita del usuario
7. Crea todo en status PAUSED
8. Reporta IDs creados
