# Vibiz — Arquitectura de Producto

> Objetivo: mapear las ~40 MCP tools y 5 slash commands de Vibiz para extraer **el blueprint de organización de features** que vamos a aplicar a Agencia Agéntica.

## Slash commands conocidos

| Comando | Propósito declarado | Notas tras uso |
|---|---|---|
| `/vibiz:status` | Estado actual del proyecto / cuentas conectadas | — |
| `/vibiz:onboard` | Flujo de onboarding inicial | — |
| `/vibiz:post [sha]` | Borrador de post (asociado a commit opcional) | — |
| `/vibiz:ad [image\|video]` | Crear creativo de anuncio | — |
| `/vibiz:launch` | Lanzar campaña | — |

## Categorías declaradas de MCP tools

1. **Generación** — imágenes, vídeos, UGC, funnels
2. **Social** — publishing, scheduling
3. **Inbox** — DMs, comments
4. **Analytics** — métricas
5. **Meta Ads campaign management** — gestión campañas Meta
