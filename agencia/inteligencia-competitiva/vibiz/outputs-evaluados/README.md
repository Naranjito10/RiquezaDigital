# Outputs Evaluados de Vibiz

> Muestrario de outputs reales con rating según rúbrica del README principal (1-5).

## Convención de nombres

`YYYY-MM-DD_<tipo>_rating<N>.md`

Ejemplos:
- `2026-05-25_carrusel-linkedin_rating4.md`
- `2026-05-25_post-meta_rating2.md`
- `2026-05-25_ad-creative-image_rating5.md`

## Plantilla por archivo

```markdown
# [Tipo de output]

- **Fecha:** YYYY-MM-DD
- **Plataforma objetivo:** [Meta / LinkedIn / TikTok]
- **Comando / tool usado:** [/vibiz:post / etc.]
- **Rating:** N/5 ([listo para publicar / retoque ligero / borrador / desecho / peor que cero])

## Input dado a Vibiz

[Qué le pediste]

## Output recibido

[Pegar literal — texto, descripción de imagen, etc. Si es imagen/vídeo guardarlo en subcarpeta `assets/`]

## Evaluación

- **Qué funciona:**
- **Qué falla:**
- **Qué cambiaría yo:**
- **¿Replicable con stack RD (Gemini/Canva/etc.)?:** [sí / no — esfuerzo estimado]
```
